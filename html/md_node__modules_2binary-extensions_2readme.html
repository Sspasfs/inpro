binary-extensions

‍List of binary file extensions

The list is just a JSON file and can be used anywhere.

Install

$ npm install binary-extensions

Usage

const binaryExtensions = require('binary-extensions');
console.log(binaryExtensions);
//=> ['3ds', '3g2', …]
